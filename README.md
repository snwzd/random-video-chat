# Random Video Chat

Meet random people online.

## Tech Stack
- Echo framework (backend in Go)
- HTMX (server side rendering with Go templates)
- Redis (as messaging system and database)
- Traefik (as layer 7 load balancer)

## Setup
```
docker compose up
```
